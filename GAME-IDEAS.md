# Game Ideas

Alla spel bygger på samma engine (engine.js) med Bold & Vibrant-stilen.
Varje spel är en enkel endless highscore-mekanik — lätt att förstå, svårt att bemästra.

## Timing / Rhythm

### Hole in One ✅ KLAR
Tap för att sätta power och vinkel på minigolf-slag. Ett slag per hål, banorna blir galnare. Score = antal hål klarade.

### Roof Hopper ✅ KLAR
En karaktär springer över hustak. Tap för att hoppa över gapen. Gapen blir bredare, taken smalare, vind tillkommer. Flappy Bird-DNA.

### Perfect Serve ✅ KLAR
Volleyboll/tennis-serve. Swipe för att kasta och slå. Targets på andra sidan nätet blir mindre och rör sig. Streak-baserad scoring med multipliers.

## Flick / Throw Sports

### Bin It ✅ KLAR
Kasta ihopskrynklat papper i papperskorgar. Vind, rörliga korgar, hinder. Alla fattar konceptet direkt.

### Dart Streak ✅ KLAR
Flicka pilar mot en darttavla. Bullseye förlänger din streak, missa tavlan och det är game over. Tavlan börjar röra sig, krympa, snurra.

### Skee-Ball Forever ✅ KLAR
Swipe för att rulla en boll uppför rampen. Klassiskt arkad-skee-ball men high-score-hålen krymper varje runda.

## One-Touch Reflex

### Goalie ✅ KLAR
Bollar flyger mot mål från alla vinklar. Swipe för att dyka och rädda. Hastighet och skruv ökar. Hur många kan du rädda i rad?

### Juggle King ✅ KLAR
Tap på en fotboll för att hålla den uppe. Gravitationen ökar, vind blåser, ibland två bollar samtidigt. Klassisk keepie-uppie.

### Punch Out ✅ KLAR
Boxing-dodge. Swipe för att ducka slag som kommer mot dig. Hastigheten ökar tills du blir träffad. Satisfying near-misses.

## Stacking / Precision

### Tower Stacker ✅ KLAR
Block svingar över skärmen, tap för att droppa. Överhäng skärs bort. Se hur högt du kan bygga innan det blir för smalt. Bevisad viral mekanik.

### Pint Pour ✅ KLAR
Luta för att hälla öl/läsk i glas. Fyll till linjen utan att det rinner över. Glasen krymper, rullbandet snabbar på. Oväntat satisfying.
